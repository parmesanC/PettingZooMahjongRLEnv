## 测试步骤

### 1. 启动本地HTTP服务器

```bash
# 进入项目目录
cd src/mahjong_rl/web/phaser_client

# 启动Python HTTP服务器
python -m http.server 8080
```

### 2. 打开浏览器访问

在浏览器中打开: `http://localhost:8080/index.html`
